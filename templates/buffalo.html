Scan a File

File must exist in S3 under encrypted/{type}/{filename}

How Pa*(b3) Works
  1. Download encrypted bytes from S3
  2. SHA-256 → 64-char hex hash
  3. Convert hash → 32-float vector
  4. Cosine similarity vs normal profile
  5. Fitness ≥ threshold → MAAA
  6. Fitness < threshold → WAAH

Eq. 2 — Pa*(b3)
Pa*(b3) = Σ(normal, attack_fitness)

Cosine similarity between file's hash vector and the normal profile. Identifies whether a file's hash pattern matches expected normals.

Eq. 3 — Na*(b3)
Na*(b3) = b3(normal) − b3(attack)

Separation between normal and suspicious fitness. Confirms anomaly magnitude and determines action: BLOCK, FLAG, or ALLOW.
